Aula 6 – More Than One 🔢

1️⃣ Explore – These & Those

Agora vamos falar sobre mais de um objeto.

These → estes / estas (objetos próximos)

Those → aqueles / aquelas (objetos distantes)

Compare:

This is a computer.

These are computers.

That is a chair.

Those are chairs.

2️⃣ Singular x Plural

computer → computers

keyboard → keyboards

screen → screens

table → tables

👉 Regra básica: adicionamos S no final.

This is a computer.

These are computers.

3️⃣ Practice

These are my computers.

These are my keyboards.

Those are chairs.

Those are tables.

4️⃣ Listening

Alex: These are my computers.

Alex: Those are my chairs.

Estes são meus computadores. Aqueles são minhas cadeiras.

5️⃣ Game 🎮

Escolha a resposta correta:

👀 (objetos perto)

👀 (objetos longe)

6️⃣ Mission – Visual Description 👁️

Olhe ao seu redor e fale:

Exemplo:
