Aula 7 – Articles in real life 🧠

1️⃣ Explore – a / an / the + Reading

📚 Vamos aprender usando uma história:

I am a student in a technology class.

I have a computer and an internet connection.

The computer is fast and the internet is stable.

I open a project and write a code.

The code is simple, but the result is good.

I fix an error and test the program.

The program works and I feel like a developer.



🎯 Sua missão: Traduza esse texto

📌 Regras simples

a → usado antes de palavras com som de consoante

a computer, a student

an → usado antes de som de vogal

an error, an internet connection

the → algo específico

the computer (um computador específico)
